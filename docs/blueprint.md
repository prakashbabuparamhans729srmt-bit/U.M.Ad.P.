# **App Name**: UCLAP Admin Panel

## Core Features:

- Dashboard View: Display key metrics such as pending orders, processing orders, cancelled orders, completed orders, and total orders in a clear and concise manner.
- Order Statistics: Show a breakdown of product order statistics, including the number of pending, processing, cancelled, and completed orders.
- Sales Overview: Provide an overview of total sales, including the total amount.
- Category Management: Enable the admin to manage main categories, subcategories, and child categories, and add new categories as needed.
- Banner Management: Allow the admin to manage banners, including adding, editing, and deleting banners.
- Timeslot and Date Management: Enable the admin to manage timeslots and dates for service availability.
- Partner Section Management: Allow admin to see and edit partner related data

## Style Guidelines:

- Primary color: Light gray (#F0F0F0) for a clean and professional look.
- Background color: Almost White (#FAFAFA), which is slightly lighter than the primary, for clear content separation.
- Accent color: Pale Blue (#D0E8EF) for interactive elements and highlights, to bring some contrast.
- Body and headline font: 'PT Sans' (sans-serif) for a modern and readable interface.
- Simple and professional icons for easy navigation.
- Clean and organized layout with a sidebar for navigation and a main content area for displaying data.
- Subtle animations for transitions and interactions to improve user experience.
